# 介绍

Speed Components 是一个基于 Ant Design Vue 的组件库，提供了一系列高质量的 Vue 组件，帮助开发者快速构建企业级应用。

## 特性

- 🚀 基于 Vue 3 和 Ant Design Vue 4
- 📦 开箱即用的高质量组件
- 🔍 完整的 TypeScript 支持
- 🔧 详细的文档和示例

## 浏览器支持

- 现代浏览器
- Chrome >= 80
- Firefox >= 78
- Safari >= 13
- Edge >= 80

## 版本要求

- Vue >= 3.4.0
- Ant Design Vue >= 4.1.0
